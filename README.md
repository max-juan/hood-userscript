# Hood Assistente de Concessao - Userscript

Userscript do Tampermonkey que ajuda a mesa de credito da Robbin a gerar pareceres padronizados a partir do Retool.

## Instalacao (analistas)

1. Instale a extensao **Tampermonkey** no seu navegador:
   - [Chrome](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
   - [Edge](https://microsoftedge.microsoft.com/addons/detail/iikmkjmpaadaobahmlepeloendndfphd)

2. Clique aqui para instalar o script:
   **[👉 INSTALAR HOOD ASSISTENTE](https://github.com/max-juan/hood-userscript/raw/main/hood-assistente.user.js)**

   O Tampermonkey vai abrir uma aba pedindo confirmacao. Clique em **Instalar**.

3. Pronto! Acesse uma analise de credito no Retool e o botao laranja **R** aparece no canto inferior direito.

## Atualizacoes automaticas

O script atualiza sozinho: o Tampermonkey verifica diariamente se ha uma nova versao no repositorio. Voce nao precisa fazer nada.

Para forcar atualizacao imediata: Tampermonkey > Painel > clique em "Verificar atualizacoes" no script.

## Funcionalidades

- Extrai dados da tela de analise (empresa, motor, SCR PJ/PF, parceiro)
- Busca dados de compras na planilha do parceiro (Malwee, Moura, Brinox)
- Calcula multiplicadores conforme a politica de credito vigente
- Gera parecer padronizado pronto para copiar
